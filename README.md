# 🚀 100 Days of MLOps - Progress Tracker

Welcome to my repository documenting my journey through the [KodeKloud 100 Days of MLOps](https://engineer.kodekloud.com/signup?referral=6890efd7717ac8036fba9dcb) challenge!

## 🎯 Objective
As a professional with a strong foundation in DevOps, infrastructure orchestration, and Kubernetes, I am using this challenge to expand my expertise into Machine Learning Operations (MLOps). This repository serves as a public portfolio of my daily hands-on tasks, scripts, environment configurations, and automation workflows.

## 🛠️ Tech Stack & Tools
Throughout this 100-day journey, I will be working with industry-standard tools to build a complete MLOps ecosystem:
* **Languages & Frameworks:** Python, Bash
* **Environment & Automation:** Makefiles, Virtual Environments (`venv`, `uv`), Pre-commit hooks
* **Containers & Orchestration:** Docker, Kubernetes (K8s)
* **MLOps Core:** DVC (Data Version Control), MLflow, FastAPI, BentoML, Argo Workflows

## 💡 How to Navigate this Repository
Each completed day has its own dedicated folder (e.g., `day-01/`, `day-02/`). Inside each folder, you will find:
1. The scripts or configuration files required to solve the task.
2. A `README.md` specific to that day, explaining the approach and the underlying concepts.
